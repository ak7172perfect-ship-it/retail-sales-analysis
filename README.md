# Retail Sales Analysis & ETL Pipeline

## Overview

This project analyzes retail sales data using an end-to-end ETL pipeline. The objective is to extract, clean, transform, and visualize data to generate meaningful business insights.

## Tools and Technologies

* Excel (Data Source)
* Python (Pandas for Data Cleaning and Automation)
* MySQL (Data Storage and Querying)
* Power BI (Dashboard and Visualization)

## ETL Pipeline

* Extracted raw data from Excel
* Cleaned and transformed data using Python
* Loaded processed data into MySQL
* Built interactive dashboards in Power BI

## Key Insights

* West and East regions contribute over 60% of total sales
* Technology is the top-performing category
* Consumer segment generates the highest sales (~50%)
* Sales peak in November–December (holiday season)
* Profit is lowest in July

## Files Included

* sales_etl_pipeline.py – Python automation script
* retail_analytics_db.sql – SQL queries
* superstore_sales_analysis.xlsx – dataset
* dashboard.png – Power BI dashboard screenshot

* Automated data cleaning and transformation processes
* Generated actionable business insights
